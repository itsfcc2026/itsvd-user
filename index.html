ตั้งค่า Web App URL

วาง URL จาก Google Apps Script ที่นี่

วิธีรับ URL (ทำครั้งเดียว)

  1. เปิด Google Sheet → คลิก "ส่วนขยาย" → "Apps Script"
  2. วางโค้ดจากระบบ → กด 💾 บันทึก
  3. กด "ทำให้ใช้งานได้" → "การทำให้ใช้งานได้ใหม่" → เลือก "เว็บแอป"
  4. ตั้ง "ดำเนินการในฐานะ: ฉัน" และ "ผู้ที่มีสิทธิ์เข้าถึง: ทุกคน"
  5. กด "ทำให้ใช้งานได้" → คัดลอก URL ที่ได้มาวางด้านล่าง
